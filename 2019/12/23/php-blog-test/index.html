主要有以下几个功能

  2. 用户注册
  3. 用户留言
  4. 用户名与用户密码的更新

贴一份链接好了：a php blog

主要思路

  1. 先建个数据库，2个表分别用来存储用户信息和留言板信息
  2. 建立文件用以链接数据库
  3. 写主页，主要能显示登录登出以及留言功能
  4. 登录后自动调转至user界面，可进行登出，留言与更新操作

安全相关

  1. htmlspecials()该加的都加了（也许？
  2. 预防sql注入：使用PDO语句(以login.php为例)

PDO原理：在传入参数之前已经构建好了语法树，所以后期语法结构并不会被改变，传入的参数只会被当作参数看待
